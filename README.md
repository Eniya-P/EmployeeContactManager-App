## App Description
This Android application helps manage employee details such as name, ID, phone and department. 
It provides options to call, send SMS and send email directly from the app.
Developed using Java and Android Studio using ListView, CardView and Intent features.

## Features
- Add employee details
- Call employee
- Send SMS
- Send Email
- ListView display
